## 実験管理について

Hydraを使って実験管理してみる
- yamlによる構造化したパラメータ管理
- 実行ごとに設定と結果を自動保存
- コマンドラインからパラメータ上書き
- 実験条件の切り替えを簡単に

**Hydraはコードを触らずに実験条件を切り替えられる！**

通常は以下の感じで整理するといいらしい
```
.
├── conf
│   ├── config.yaml          # 全体の司令塔
│   └── model                # モデルごとの設定を入れるフォルダ
│       └── lightgbm.yaml    # 今回作ったファイル
└── train.py                 # 実行するPythonファイル
```

conf/config.yaml でどのモデルを設定を使うかを決める
